# 烦人的村民-哨塔建筑包

## 来源说明
尝试在线检索“烦人的村民常见建筑类型”时，当前运行环境无法稳定返回搜索页面内容。
本数据包基于整合包现有玩法风格，先提供一个可直接落地的“村民哨塔”建筑。

## 使用方法
1. 进入世界后执行 `/reload`。
2. 执行：`/function av_build:spawn_watchtower`
3. 会在你附近生成一座可攀爬的哨塔。

## 目录
- `data/av_build/functions/spawn_watchtower.mcfunction`：建筑生成逻辑
- `data/av_build/functions/load.mcfunction`：载入提示
