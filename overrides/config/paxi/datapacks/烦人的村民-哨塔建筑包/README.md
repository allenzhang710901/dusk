# 烦人的村民-哨塔建筑包

## 本次补全内容（你说的“还有啥没做”）
此前只做了单体哨塔，现在补上了更完整的“烦人的村民前哨营地”建筑组合：

- 哨塔：`/function av_build:spawn_watchtower`
- 兵营：`/function av_build:spawn_barracks`
- 铁匠铺：`/function av_build:spawn_blacksmith`
- 一键组合：`/function av_build:spawn_outpost_set`

## 使用方法
1. 进入世界后执行 `/reload`。
2. 任选上面命令执行，建筑会在你附近生成。

## 目录
- `data/av_build/functions/spawn_watchtower.mcfunction`：哨塔
- `data/av_build/functions/spawn_barracks.mcfunction`：兵营
- `data/av_build/functions/spawn_blacksmith.mcfunction`：铁匠铺
- `data/av_build/functions/spawn_outpost_set.mcfunction`：一键前哨营地
- `data/av_build/functions/load.mcfunction`：载入提示

当前运行环境无法稳定返回公开搜索引擎页面，所以没有可靠抓取到统一的“官方建筑清单”。
